Instruksi Penggunaan:

Ekstrak seluruh file dalam RAR ini ke suatu lokasi di komputer Anda

Kopi seluruh file yang ada di dalam folder Library Files

Buka folder library Proteus 8 Anda, biasanya terdapat di folder instalasi (...\Labcenter Electronics\Proteus 8 Professional\DATA\LIBRARY)

Tempel seluruh file yang dikopi tadi

Kembali ke lokasi ekstraksi folder

Buka file beta.pdsprj

Klik kanan sensor Ultrasonic

Pilih edit properties

Pada kolom UltraSonicSensor.HEX arahkan ke tempat hasil ekstraksi folder Library Files tadi

Pilih UltraSonicSensor.HEX

Tutup tab edit component

Klik kanan modul Rain Sensor

Pilih edit properties

Pada kolom Program File arahkan ke tempat hasil ekstraksi folder Library Files tadi

Pilih RainSensorsTEP.HEX

Tutup tab edit component

Buka folder Arduino_Code

Buka file BETA.ino

Klik File kemudian Preferences

Centang kotak compilation dan upload

Tutup tab Preferences

Klik tombol verify (lambang centang di pojok kiri atas layar) tunggu sampai compile selesai

Pada terminal, cari teks bertuliskan "Linking everything together..." kemudian cari alamat file dengan ekstensi *.HEX kemudian kopi alamat tersebut

Kembali ke Proteus

Klik kanan Arduino Uno R3

Pilih edit properties

Pada kolom Program File tempelkan alamat yang dikopi tadi

Tutup tab edit component

Jalankan simulasi dengan menekan tombol Play pada pojok kiri bawah aplikasi Proteus 8
